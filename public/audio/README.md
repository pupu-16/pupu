# Atmosphere Radio — Audio Files

Place 5 ambient audio files here. Each should be 30-60 seconds, loopable, low volume, no vocals.

## Required files

| File | Channel |
|------|---------|
| `rain-room.mp3` | Rain Room |
| `midnight-lofi.mp3` | Midnight Lo-fi |
| `soft-piano.mp3` | Soft Piano |
| `ambient-space.mp3` | Ambient Space |
| `morning-light.mp3` | Morning Light |

## Format

- MP3 (all browsers supported)
- 30-60 seconds
- Designed for seamless looping
- Low volume, ambient, no sudden sounds
- Royalty-free or self-created

## Sources for free ambient audio

- Pixabay (pixabay.com/music)
- Freesound (freesound.org)
- Mixkit (mixkit.co/free-stock-music)

## If files are missing

PUPU will still work — the Radio UI will function in Mock mode
(visual playback indicator only). A subtle hint will appear in the
Music Drawer noting that audio files haven't been added yet.
